### Welcome to Hunter's ePortfolio

**Self Assessment:**
Working through the Computer Science program at SNHU and building my ePortfolio has helped me see how much I’ve grown in both technical skills and overall problem solving. Even though my long term goal is to work in sales, the coursework pushed me to think more logically, communicate clearly, and break down complex problems, which are skills that matter in any professional setting. The capstone especially helped me pull everything together, because I had to take an existing project, improve it multiple times, and explain my decisions in a way that someone else could understand. That process made me more confident in presenting my work and talking through technical ideas with different audiences.

Throughout the program, I’ve learned how to appropriately collaborate in teams, communicate with stakeholders, and apply concepts like data structures, algorithms, software engineering, database design, and security. These experiences helped me understand how real systems are built and why clear communication matters just as much as the code itself. The artifacts in my portfolio show this progression: one focuses on software design, another on algorithms and data structures, and the last on database integration.
